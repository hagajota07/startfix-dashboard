# Startfix · Painel de Gestão

Painel interno de gestão semanal — Hagajota Studio.

- 5 abas: Dashboard, Registrar Semana, Campanhas, Semana 1, Histórico
- Persistência local (`localStorage`) — nenhum backend
- 4 semanas de baseline pré-Hagajota carregadas automaticamente
- Reset manual via botão `⟲` no header

**Ritual:** toda segunda 9h, abrir, registrar a semana anterior e decidir 1 ajuste.
